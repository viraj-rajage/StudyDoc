Title:- How To Add Color In C Language Output (Terminal)
In Following Program 1st Create cyan function in this print the cyan color for that you need to write cyan color code in printf function and in reset function you need to write color code for reset the color for ex. use call cyan() at line no. 4 and you call reset() function in line no 20 then in 4 to 20 line of code can get cyan color 
#include<stdio.h>

#include<stdlib.h>
/*

more color code:
    1> Green ->  \033[0;32m  
    2> Purple ->  \033[0;35m 
    3> Red ->  \033[0;31m
    4> Blue ->  \033[0;34m
    


*/
void cyan()
{
    printf(" \033[0;36m");
    // for cyan color
    // \033[0;36m this is the color code of cyan color
}

void reset()
{
    printf("\033[0m");
    // for cyan color
    // \033[0m this use for reset the color means at this point only user color apply
}

int main(){

    cyan();
    printf("StudyDoc Creation");
    printf("StudyDoc");
    reset();
    printf("Default color");

    return 0;
}
